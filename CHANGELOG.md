# Changelog

All notable changes to this project will be documented in this file.

The format is inspired by Keep a Changelog, and this project uses semantic versioning principles where practical.

## [1.0.0] - 2026-04-22
### Added
- Initial public release of Everly web application.
- Java Servlet/JSP layered architecture with Front Controller and authentication filter.
- Memory management, collaboration, groups/events, journaling, vault/recycle flows.
- Public feed with profile, post, follow/block, like/comment/report capabilities.
- Service-layer media encryption and key metadata handling.
- PostgreSQL schema, migrations, and configuration templates.
